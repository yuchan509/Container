## 환경 구성(Docker, Docker-Compose, kubectl, kustomize, minikube)
```shell
# 목록 확인
ls

# shell script 만들기.
$ cat > {shell script name}.sh

# 권한 부여.
$ chmod u+x {shell script name}.sh

# 스크립트 실행.
./{shell script name}.sh
```
